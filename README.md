# XKE Frontend
Web application for XKE slots management.

## Dependencies
- npm
- bower (`npm install -g bower`)
- grunt-cli (`npm install -g grunt-cli`)

## Install
- `npm install`
- `bower install`

## Build
- `grunt build`

## Deploy ([GitHub Pages](http://xebia-france.github.io/xke-front))
- `grunt deploy`
